Treniruotės vedlys – PWA prototipas

Kas tai yra:
- Paprasta telefono programėlė treniruotės eigai sekti.
- Pasirenkate asmenį: Kristijonas arba Vilmantas.
- Pasirenkate treniruotę: A, B arba C.
- Matote vieną pratimą, tris setus, pakartojimus ir planuotą svorį.
- Kiekvieną setą galima pažymėti kaip atliktą. Pažymėtas setas tampa žalias.
- Duomenys saugomi telefone / naršyklėje per localStorage.
- Programėlė veikia ir be interneto, jeigu yra paleista kaip PWA iš HTTPS svetainės.

Kaip įsidiegti į Android:
1. Įkelkite visą šį aplanką į HTTPS svetainę, pvz., Netlify, GitHub Pages ar kitą statinį hostingą.
2. Telefone atidarykite svetainės adresą su Chrome.
3. Chrome meniu pasirinkite „Add to Home screen“ arba „Install app“.
4. Programėlė atsiras telefono pradžios ekrane.

Pastaba:
Atidarius index.html tiesiogiai iš failų sistemos, puslapis veiks kaip paprastas puslapis, bet tikras PWA įdiegimas gali neveikti. PWA diegimui reikia HTTPS arba localhost.
